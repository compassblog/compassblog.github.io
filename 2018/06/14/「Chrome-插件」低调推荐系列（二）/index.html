「Chrome 插件」低调推荐系列（二）

发表于 2018-06-14 | 分类于 Chrome

喜欢 Chrome 的朋友都知道，扩展插件占据 Chrome 浏览器 50% 以上的功能，我也坚信很多人跟我一样是因为「扩展功能」爱上 Chrome 的，上周低调了推荐了几款我高频率使用的 Chrome 插件：

「Chrome 插件」低调推荐系列（一）

那么问题来了，为什么今天还要推荐呢？原因非常简单，今天公众号增加了 100 多个读者，这对我来说很多了，而且后台经常是很多人问我有没有什么高效好用的工具推荐的，尽管不可能每个人都如我一般钟情于 Chrome，但有用的高效率的工具，有机会还是要分享出来的。

我们先大致罗列下前面已经极力推荐过的插件：「谷歌访问助手」「油猴插件」「Listen1」「印象笔记」「Octotree」……

这些插件大都是一些高效好用的插件，具体功能就不过多赘述了，需要的朋友可以翻下前面的文章，里面也提供了这些插件的获取方式。好了，下面就是今天推荐的插件：

1、微博图床

现在越来越多的人使用 Markdown 来写文章了，而在 Markdown 文章中插入图片通常情况下都会使用图床。本人最经常使用的图床是七牛云，但是登录上传复制链接实在麻烦，后来发现 QQ 空间相册也可以做图床，但有限制，当然，也用过 GitHub 作为图床存放图片。

以前写文章放的图片不多还没什么感觉，前两篇文章因大量使用图片才发现，我急需一个好用的图床工具啊，说时迟那时快，立刻想到 Chrome 插件，找到了「微博图床」这个超级好用的图床插件。

「微博图床」这款插件可以帮助我们意见将我们需要保存的图片上床到微博，然后直接提供 Markdown 链接，而且还包括网页链接和 HTML 语法链接，效率奇高，屡试不爽，好不夸张的说，我已经离不开微博图床了，如下图所示：

获取方式：直接在 Chrome 商店搜索「微博图床」即可获取。

PS：这里再提醒下，相信前面介绍过的「谷歌访问助手」插件，尽管不会科学上网，也足够大家很顺畅的使用 Chrome 商店了。

2、Adblock Plus

这是我今天发现的一款插件，看上面介绍相信大家就已经迫不及待想试试这款插件的了，「Adblock Plus」是一款广告拦截的插件，号称全球超过 5 亿人次下载量，是全球最受欢迎的广告拦截软件之一。 大家可能都已经受够了看一个短短的视频竟 2 分钟的广告了，各大视频网站现在的广告也确实是很长啊，而且现在经常动不动就 180 多秒，不过有了这款插件，亦可以完全跳过广告了。当然，有了这款插件，各种弹窗广告也是必杀的。

而且更不可思的是，你要是喜欢某个广告，还可以选择性的设置个白名单来看广告，当然啦，估计喜欢广告的朋友不多。虽然前面介绍的「油猴插件」也可以安装个脚本来去广告，但那个略麻烦些，这个就真的是超强了，我刚试了下，竟然强到把视频下面的评论都杀掉一截，真是有点瑟瑟发抖，如下图所示：

获取方式：直接在 Chrome 商店搜索「Adblock Plus」即可获取。

3、右键搜

如上图所示，「右键搜」这款扩展插件号称超过300种功能，可以方便的选择、组合或自定义页面、划词、图片菜单，并有生成、解析二维码和短网址功能，也可以设置可使用云存储，最新增加功能共享、资源下载、重命名等功能。右键搜伴侣增加超级拖拽。

但是我经常用的功能只有 2 个，一个是选中文字右键直接翻译，另一个功能是选中文字生成二维码，选中文字生成二维码这个功能我还是蛮喜欢的，如下图所示：

获取方式：直接在 Chrome 商店搜索「右键搜」即可获取。

4、infinity 新标签页

这是我超级超级喜欢的一款插件，原因有三：一是新建标签页面时，打开的是一个超级唯美的页面，不用再面对一个空旷旷的页面，；二是可以同步笔记和待办事项；三是在新的标签页可以自定义常用的网站，并且配以超好看的图标和壁纸，更为重要的是，壁纸随时可换。如下图所示：

获取方式：直接在 Chrome 商店搜索「infinity」即可获取。

5、Momentum

「Momentum」这款插件的主要功能主要也是在新建标签页时，呈现出一个唯美的页面，让你不用再面对一个空旷旷的页面的，功能与「infinity」相似，全英文页面，自动识别时刻并提醒，功能虽然没有「infinity」强大，但还是推荐下，如下图所示：

获取方式：直接在 Chrome 商店搜索「Momentum」即可获取。

6、扩展管理器

最后再推荐一款插件吧。相信很多与我一样过于依赖 Chrome 插件的朋友都有这样的烦恼，插件太多管理起来就有点麻烦，但有不可全部都开，这时候该怎么办呢？有 Chrome 在，不要方。

「扩展管理器」这款插件可以帮助我们一键管理所有扩展，快速开启/禁用、批量闪电管理，智能排序，右键卸载、锁定、选项配置，角标提醒，大小布局随心配，快捷、简单、安全。 如下图所示：

获取方式：直接在 Chrome 商店搜索「扩展管理器」即可获取。
