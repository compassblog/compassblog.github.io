Markdown 编辑器使用指南

1、快捷键

  • 加粗： Ctrl/Cmd + B
  • 标题： Ctrl/Cmd + H
  • 插入链接： Ctrl/Cmd + K
  • 插入代码： Ctrl/Cmd + Shift + C
  • 行内代码： Ctrl/Cmd + Shift + K
  • 插入图片： Ctrl/Cmd + Shift + I
  • 无序列表： Ctrl/Cmd + Shift + L
  • 撤销： Ctrl/Cmd + Z

2、常用语法

（1）、标题

语法格式： ‘#’+’空格’+’文本’

一级标题：一个 # 号

二级标题：两个 ## 号

三级标题：三个 ### 号

四级标题：四个 #### 号

五级标题：五个 ##### 号
六级标题：六个 ###### 号

（2）、列表

无序列表语法格式： ‘-‘ + ‘空格’ ＋ ‘文本’

  • 文本一
  • 文本二
  • 文本三

有序列表语法格式： ‘数字’ + ‘.’ + ‘空格’ + ‘文本’

  1. 文本一
  2. 文本二
  3. 文本三

（3）、链接

在 Markdown 中插入链接不需要其他按钮，你只需要使用［显示文本］(链接地址)这样的格式语法即可。例如：

不添加显示文本方式（没有关键字文本，直接显示链接）：’[]+(https://compassblog.github.io/)'

添加显示文本方式（链接被隐藏，只显示关键字文本）：’ [指南者驿站]+(https://compassblog.github.io) ‘

指南者驿站

（4）、图片

插入图片的语法与插入链接的语法很像，只是前面多了一个 !.语法如下：
![图片的标注](图片链接地址)

例如：!+[]+(http://p5xem2laz.bkt.clouddn.com/about/wechat-qcode.jpg)

（5）、引用

语法： ‘>’+’空格’+’文本’
